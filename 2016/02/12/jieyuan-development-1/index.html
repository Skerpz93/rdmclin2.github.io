结缘的功能设想-初步


恩，结缘的宗旨是帮助程序员们找到一个适合自己的好妹子，以及帮助想养猿的妹子找到一个靠谱的程序猿。
– 《结缘的宗旨yy版》

用户群体为程序猿，妹子。性别为男，女。设想的用户字段如下:

  • name 姓名
  • username 用户名
  • email 邮箱
  • gender 性别: {male,female}
  • phoneno 电话号码
  • location 所在地
  • password 登录密码
  • url 个人网站地址
  • avatar 头像地址
  • signature 个人签名
  • create_at: 创建日期
  • state: 状态: {single,inlove}
  • accessToken: access token
  • friends [] 好友列表

目前设想的功能有

  • 用户登录/注册
  • 个人主页
  • 聊天功能

使用的技术栈为

  • Node.js 后端
  • MongoDB 数据库
  • Vue.js 前端
