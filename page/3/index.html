重装系统

网络，TimeMachine

1）关闭Mac系统
2）同时按下组合键 command +R
3）再按下开关键 ，command +R不能松开
ps：使用网络恢复系统由于网络原因时间较长，如果没有TimeMachine建议制作启动u盘,另强烈建议制作TimeMachine备份。

制作启动u盘

教程这篇文章说的比较详细了，不赘述。做完u盘，开机按住option键，选择制作的u盘即可，一般半小时内完成。
