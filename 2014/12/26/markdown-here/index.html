    为什么要用markdown格式编写邮件？

    • 逼格
    • 对于习惯使用markdown的人群来说，markdown可能是写作的唯一方式。

    什么是markdown here？

    markdown here是一款浏览器插件，支持chrome，firefox，safari，opera等浏览器，以及thunderbird，postbox和icedove等邮箱客户端（很遗憾在foxmail中还无法使用）。通过这个插件可以让你在邮件中使用markdown。【ps：markdown here也支持evernote的网页版】

    【More info】插件官网

    使用教程

    插件获取及安装

    以chrome插件为例。

    • 如果已经翻墙，从【官网下载地址】直接在chrome应用商店中安装chrome插件。
    • 如果你被墙困住了，可以看看这篇文章：红杏出墙。
    • 如果被墙困住了也不想翻，可以直接下载插件扩展,如果在天朝你还不会在chrome中安装插件，可以看看这篇教程。

    安装完成后在chrome的插件栏中可以看到紫色的蝴蝶图标。

    markdown here

    这里以在chrome中用gmail编写邮件为例，解释如何使用markdown-here插件。

    • 首先打开Gmail【当然前提是gmail没有被墙】
    • 点击写邮件
    • 在富文本区域编写markdown代码。
    • 点击上面提到的蝴蝶进行转换，或者右键菜单进行markdown转换，也可以快捷键ctrl-alt-M。当然能转换过去也能再转换回来继续编辑。
