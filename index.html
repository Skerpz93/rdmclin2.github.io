我的前后端学习之旅[updating]

发表于 2015-06-26   |  

为什么要学习前后端开发

javascript目前是github上最火的语言，可以在githut上查看github的
语言指数,也可以到githubranking上看排名靠前的
库的star数量，几乎都和js，前端有关。在学习ionic的过程中接触到了非常多的前端后端开发的概念，
每一次看到优秀的js库都让我心潮澎湃，不禁想记录下来自己的学习路线，接触到的开源项目等。之所以
说是前后端是因为nodejs的出现让js具备了进军后端的能力，而[angularjs]
(https://angularjs.org/)让web进入mobile developing的领域，
另外electron让web能够访问系统资源，催生了atom,slack等一批native web app的诞生。
因此本屌觉得javascript潜力无限，未来定是web的世界，是时候学习一波web开发了。

PS:开始写这篇博文的时候刚接触前端开发1个月左右，言语多有不当之处请指出。

编辑器推荐

写代码，没有一款适合自己的编辑器怎么行?

    atom试用

  • vscode
    $: 微软刚出的web编辑器,和atom一样基于electron,不过速度快，性能强劲,后起之秀

  • javascript语言精粹 $: javascript 精华，需反复阅读

社区

多接触社区，可以了解到最新的技术潮流，还能结识好朋友

  • cnodejs社区,非常有爱的nodejs社区
  • angularjs中文社区
  • 前端乱炖

优秀开源项目

js有很多优秀的开源项目

  • velocity $: Accelerated JavaScript animation.
  • io.js $: io.js是joyent/node项目的一个fork，
    关于它和node的关系可以看io.js和node.js怎么了？
  • node $: evented I/O for v8 javascript
  • impress.js $: 你喜欢prezi吗？impress由此而来
  • nw.js $: 原node-webkit
  • electron $: atom的外壳程序（atom是原子,electron是电子)
  • bower $: web开发的包管理工具
  • jade $: nodejs 模版引擎
  • PM2 $: Node.js / io.js 进程管理工具，可视化工具keymetrics
  • Mongodb $: 非关系型数据库,使用json格式,nosql
  • Mongoose $: 异步环境下的Mongodb对象建模工具
  • angular.js $: 谷歌出品js前端框架，TDD敏捷开发，前端依赖注入
  • express $: tj大神的作品，node应用框架，如其名，快速开发应用
  • bootstrap $: github上star最高的项目，twitter出品，前端开发框架

  • hexo $: nodejs 博客框架，本博客即用的hexo
  • coffeescript $: JavaScript good parts,编译出javascript代码,多语法糖
  • gulp $: Automate and enhance your workflow 工作流构建自动化
  • grunt $: The JavaScript Task Runner ,基于Node.js的项目构建工具
  • stylus $: css语言 for nodejs
  • reveal.js $: The HTML Presentation Framework HTML演示框架

Eclipse 安装 Fatjar.jar失败的解决方法

发表于 2015-05-10   |   分类于 dev   |  

Fatjar简介

Fatjar.jar是eclipse上很方便进行打包的插件，对于较小的项目使用eclipse内置的打包jar可能就够了，但是如果工程较大，需要引用资源之类的，用fatjar比较方便省心。不过笔者在安装的时候遇到不少困难，不管是在线方式，还是离线方式都无法安装成功，现在纪录一下。在Windows => prefernce => fat jar preference 看到他就说明已经安装成功。

Fatjar在线安装

help －> Install New Sofware －> Work with。
粘贴 http://kurucz-grafika.de/fatjar
然后回车，等待安装即可,时间较长

Fatjar手动安装

在sourceforge上手动下载最新版本（目前0.31），解压，将.jar文件复制粘贴到
$eclipse_Home\plugins文件夹下，重启eclipse即可。

Android Studio 如何彻底删除Project

发表于 2015-03-02   |   分类于 dev   |  

缘起

之前试用过Android Studio，但是因为要删除一个Project总是删除不掉，就算能delete掉内容，目录却怎么也删不掉，抓狂了，卸了。不过Android Studio毕竟是安卓开发的未来，现在出了正式版，所以又来试用了，这次这个问题在stackOverFlow上得到了解答。

方法

点击 File -> Close Project。然后在文件目录中切到AndroidStudioProjects，在其中删除这个Project的文件夹即可，简单不？
