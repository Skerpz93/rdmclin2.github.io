重装系统

－ 网络，TimeMachine
1）关闭Mac系统
2）同时按下组合键 command +R
3）再按下开关键 ，command +R不能松开
ps：使用网络恢复系统由于网络原因时间较长，如果没有TimeMachine建议制作启动u盘,另强烈建议制作TimeMachine备份。

  • 制作启动u盘
    教程这篇文章说的比较详细了，不赘述。做完u盘，开机按住option键，选择制作的u盘即可，一般半小时内完成。

iTerm2配色

安装完oh my zsh后你可能对iterm2默认的配色不太满意，可以到iTerm2-Color-Schemes将此工程克隆下来,然后在iterm2下用cmd+i呼出颜色配置，然后Colors -> Load Presets -> import 选项将schemes文件夹下面的配置都倒入进去，然后一个个试哪一款配色适合你，我选择的是kibble。

sublime2 配置

设置macdown语法高亮 ,使用Monokai Extended这个主题。

  1. 首先安装package control
  2. 输入cmd + shift + P 调出 package control,输入install package,回车
  3. 输入Monokai Extended，回车，安装完成
  4. 在Preferences -> Color Scheme -> Monokai Extended中选择一款自己喜欢的配色
