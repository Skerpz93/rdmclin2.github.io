Nexus6 root及安装xposed框架

本文为Mac系统下为Nexus6安卓6.0.1安装Xposed框架的过程。

root

  1. 要安装Xposed框架，首先机器需要root,点击下载root工具包

root工具包的不同机型可以在autoroot上找到

  1. 连接mac和手机,adb devices确定手机与电脑相连

  2. 进入刷机界面 adb reboot bootloader

  3. 命令行进入解压下载的工具包中，chmod +x root-mac.sh，然后./root-mac.sh,手机出现红色android,等待重启,查看是否已安装SuperSu权限管理工具

刷机

  • 下载你的设备对应的twrp,比如选择的是Google Nexus6，twrp-3.0.3-0-shamu.img
  • 进入bootloader刷机界面, adb reboot bootloader
  • 输入以下指令fastboot devices判断设备是否连接
  • 刷入TWRPfastboot flash recovery twrp-3.0.3-0-shamu.img,fastboot reboot重启
  • adb reboot recovery进入刷机界面

下载xposed框架和应用

  • 官方下载地址 Xposed for Lollipop and Marshmallow，根据设备CPU型号下载对应的框架和应用,比如xposed-v86-sdk23-arm.zip和XposedInstaller_3.1.apk
  • 将xposed-v86-sdk23-arm.zippush到手机中adb push xposed-v86-sdk23-arm.zip /sdcard
  • 在twrp点击install ,选中xposed-v86-sdk23-arm.zip,然后点击flash,完成后重启
  • 然后安装XposedInstaller_3.1.apk,adb install XposedInstaller_3.1.apk,安装完成后打开应用查看是否已激活

ps: 安装的时候可能遇到系统空间不足的情况,可以用adb到system目录下删除文件,参考reddit

验证下载模块

下载钉钉抢红包模块，下载完成后需要在模块列表中激活，然后重启设备，亲测可行
