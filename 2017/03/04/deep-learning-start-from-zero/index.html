第一周

监督学习

  • 给数据和正确的答案，让机器学习
  • 分类问题: 离散预测,期待离散的取值
  • 回归问题: 连续预测,期待一个连续的取值

非监督学习

只给数据，没有答案，让机器自动区分数据的结构

  • 聚类问题: 比如谷歌新闻，相关新闻聚类在一起
  • Non-clustering: “Cocktail Party Algorithm”, 比如区分两个人的声音
