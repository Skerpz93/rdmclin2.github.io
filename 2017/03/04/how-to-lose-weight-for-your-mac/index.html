    • 如何给你的Mac瘦身

    作为一个mac乞丐版用户,存储空间仅为120G,使用下来经常捉襟见肘,常常下个电影就要清一次空间,删这个删哪个,省出个几个G出来,然后今天存储只剩2个G了…所以到底哪些东西占领了你的硬盘呢？他们是必须的吗?作为一个学生开发者，给出一些长期占据我的电脑硬盘的罪魁祸首，也许你可以获得一些灵感.

    辅助软件

    • cleanMyMac 这玩意就像使用windows时的那一票清理软件一样，可以帮你清清缓存，删除卸载软件，看看大文件啥的，不过对于清理”其他”类型的存储完全排不上用场
    • OmniDiskSweeper 神器，帮你理清你磁盘里的哪些东西占用了多少存储，并按大小排序,下面很多存储大户都是我用OmniDiskSweeper找出来的

    杀手一: Xcode - 10G

    一个Xcode安装下去10G没了，你是IOS开发者吗？如果是，默哀。如果不是，比如博主，不知道什么时候用到了Xcode,后来又不用了，就让他静静在这占着十几个G的存储不能忍…

    Xcode是存储大户，用cleanMyMac测出相关文件16G左右

    小喽喽: QQ - 1G

    QQ会缓存你的图片文件,如果你关注了一个活跃的QQ群（比如日常消息99+),常此以往QQ缓存的图片文件会越来越多.比如博主有1个G左右的图片缓存…

    解决方案: QQ -> 偏好设置 -> 图片缓存 -> 清理. 然后退出这些活跃群，如果不愿意可以定时清理缓存

    小喽喽:iBooks - 2G

    iBooks书籍通常制作精美,不知不觉下载了不少,加起来居然有2G左右了

    Office系列 - 3.2G

    Word : 1.7G
    Excel : 1.5G
    全线转到Pages和Keynote,特殊情况转同学的Windows.

    Android-NDK - 3G

    如果你是Android开发者，你可能下在Android SDK,而且你可能编译某些库而下载了NDK，然而后面没有用了。然而NDK占掉了你系统3个G的空间

    系统语音 - 3.4G

    自己的锅，之前玩mac系统语音，下载了很多不同的声音，结果占到了3.4G的存储

    打开Computer - Macintosch HD - System - Library - Speech - Voices 目录
