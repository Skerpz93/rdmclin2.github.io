    • Eclipse 安装 Fatjar.jar失败的解决方法

    #Fatjar简介
    Fatjar.jar是eclipse上很方便进行打包的插件，对于较小的项目使用eclipse内置的打包jar可能就够了，但是如果工程较大，需要引用资源之类的，用fatjar比较方便省心。不过笔者在安装的时候遇到不少困难，不管是在线方式，还是离线方式都无法安装成功，现在纪录一下。在Windows => prefernce => fat jar preference 看到他就说明已经安装成功。

    #Fatjar在线安装
    help －> Install New Sofware －> Work with。
    粘贴 http://kurucz-grafika.de/fatjar
    然后回车，等待安装即可,时间较长

    #Fatjar手动安装
    在sourceforge上手动下载最新版本（目前0.31），解压，将.jar文件复制粘贴到
    $eclipse_Home\plugins文件夹下，重启eclipse即可。

    使用方法

    项目，右键，+Build Fat Jar.
    勾选 use extern Jar-Name可以设定导出jar路径，否则生成在项目根目录
    不要勾选select Manifest file
    勾选One-JAR,next
    选择要导入的jar包
    finish

    #Fatjar安装失败的解决方法
    1.清除缓存
    新建eclipse的快捷方式，右键属性 -> 目标 后添加 -clean 选项，eclipse将清除缓存重新加载插件。

    1. 手动清除配置
      删除$eclipse_Home\configuration\org.eclipse.update\platform.xml
      该文件会自动生成。

    3.对于版本较高的eclipse，可能需要安装较低版本的eclipse的框架（笔者遇到的情况）
    通过Help -> Install New Software… -> Work with.
    选择“The Eclipse Project Updates - http://download.eclipse.org/eclipse/updates/4.4”
    然后 勾选”Eclipse Tests, Examples, and Extras” 下的 “Eclipse 2.0 Style Plugin Support”
    之后再安装fatjar.jar即可
