前言

最近用ionic做hybrid app的开发，这篇文章用来记录探索过程中遇到的方方面面。
声明：在开始这个项目之前本屌没有接触过任何web开发相关的知识

前端

总体

首先对ionic有所了解，直接看官网即可，跟着例子走一遍，有个直观的感觉。
然后前端要学的东西很多，html5，css，js等，这些可以在w3school上学习,另外可以看看知乎上的这个回答

后端

Node

现在写后端比较火的是用nodejs编写，然后nodejs有个很好的应用开发框架叫Express，这里有篇文章是教你如何从零开始搭建node,express,monggodb服务器，这里是国人翻译的版本.

LeanCloud

如果不想自己租用服务器，编写后端代码，国外有parse，国内有LeanCloud都提供后端云服务。
由于学习后端开发需要的时间比较长，本app暂时使用LeanCloud作为后端进行开发，然后专注前端。
