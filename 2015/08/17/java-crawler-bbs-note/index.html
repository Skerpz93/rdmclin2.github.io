分析过程及代码

关于爬虫

爬虫大家都知道，由一个种子页(seed)出发，分析该页，获取页面链接，加入未访问的列表中，从未访问的表中取链接，访问，重复如上步骤即可，要注意的就是要维护一个已访问的网页列表，以避免重复访问。然而这种广义的爬虫并不符合特定的情况，例如我现在要爬的bbs板块帖子，遇到具体问题，还是得具体分析。

小百合bbs爬取过程
